# GreenBlade Lawn Care Website

This is a simple static website created for a fictional lawn care business called **GreenBlade Lawn Care**.  
The project is built using basic **HTML and CSS** and includes three pages.

## Pages Included

- **Home (`index.html`)**
  - Introduction to the business
  - Highlights of services and benefits

- **About Us (`about.html`)**
  - Company background
  - Mission and values

- **Services (`products.html`)**
  - List of lawn care services
  - Sample pricing (fake data)

## Technologies Used

- HTML5
- CSS3
